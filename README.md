# URL Tracker - Production SaaS

A production-grade URL monitoring and uptime checker with real-time dashboards and email alerts.

## Tech Stack
- **Frontend**: React + TypeScript + Vite
- **Backend**: Node.js + Express + TypeScript  
- **Database & Auth**: Supabase
- **Charts**: Recharts
- **Scheduler**: node-cron
- **HTTP Checks**: axios
- **Email Alerts**: nodemailer

## Features
- URL monitoring with periodic checks
- Real-time uptime dashboards
- Historical data with charts
- Email alerts for downtime
- Public status pages
- User authentication

## Quick Start
```bash
# Backend
cd backend && npm install && npm run dev

# Frontend  
cd frontend && npm install && npm run dev
```
